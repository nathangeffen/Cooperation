\subsection{Detailed Description}
This class implements the functionality of competitors in the Prisoner's Dilemna game.

Each competitor subclasses this class. To declare a new type of competitor, use the macro REGISTER\_\-COMPETITOR and implement the decision method.

Here is how the AlwaysDefect competitor is implemented.
